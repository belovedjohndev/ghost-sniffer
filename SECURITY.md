# Security Policy

## Supported Versions

This project is maintained on the `main` branch. Security fixes are applied to the latest release only.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately so we can address it responsibly.

Preferred method:
- Email: `security@ghost-sniffer.local` (replace with a real address before public release)

Please include:
- A clear description of the issue and impact
- Steps to reproduce (minimal and safe)
- Affected versions or commit SHA
- Any relevant logs or screenshots

We will acknowledge your report within 7 days and provide an estimated timeline for a fix.

## Responsible Disclosure

We request that you:
- Avoid public disclosure until a fix is available
- Refrain from testing on systems you do not own or have explicit authorization to assess
- Provide details needed to reproduce the issue without enabling misuse

## Authorized Use Only

Ghost-Sniffer is intended strictly for authorized and ethical security testing. Use on networks or systems without explicit permission is prohibited.

## Legal and Ethical Disclaimer

This tool is provided for educational and authorized security auditing purposes only. The authors and contributors are not responsible for misuse. You are responsible for ensuring that your use complies with applicable laws and policies.
